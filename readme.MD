
### 简介
基于 Python 和 Gradio ，用于生成创作者十年变化总结表。


### 文件结构
.  
├── app.py                # 项目启动入口文件  
├── font.ttf              # 绘图所需的中文字体文件。为了hf同步，已从此文件夹删除。如需下载本地请手动添加字体文件。  
├── requirements.txt      # 项目依赖清单  
└── scripts/  
    └── word.py           # 核心逻辑实现（界面 UI 与图片生成）
